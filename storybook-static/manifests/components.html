# HomePage Composition

The HomePage is the main landing page for AirSaas, assembled from **16 section blocks** stacked vertically. Each block is a self-contained, reusable section template composed from primitive components.

## Block Inventory

| # | Block | Component | Animation | Required |
|---|-------|-----------|-----------|----------|
| 1 | Hero | `Hero` | none (above fold) | Yes |
| 2 | Logos Bar | `LogosBar` | `fade` 500ms | Yes |
| 3 | Value Proposition (light) | `ValuePropositionFrame variant="light"` | `fade-up` 700ms | Yes |
| 4 | Comparison | `ComparisonFrame` | `fade-up` 700ms | Yes |
| 5 | Feature Right | `FeatureFrame imagePosition="right"` | `fade-right` 800ms | Yes |
| 6 | Feature Left | `FeatureFrame imagePosition="left"` | `fade-left` 800ms | Yes |
| 7 | Value Proposition (dark) | `ValuePropositionFrame variant="dark"` | `fade-up` 700ms | Yes |
| 8–11 | Feature alternating | `FeatureFrame` right/left | `fade-right`/`fade-left` 800ms | Optional (repeatable) |
| 12 | Value Proposition (dark) | `ValuePropositionFrame variant="dark"` | `fade-up` 700ms | Optional |
| 13 | Testimonials | `TestimonialsFrame` | `fade-up` 700ms | Yes |
| 14 | Value Proposition (light) | `ValuePropositionFrame variant="light"` | `fade-up` 700ms | Optional |
| 15 | FAQ | `FaqFrame` | `fade-up` 600ms | Yes |
| 16 | CTA | `CtaFrame` | `scale-up` 800ms | Yes |
| 17 | Footer | `Footer` | `fade-up` 600ms | Yes |

## Animation Guidelines

All scroll animations use the `AnimateOnScroll` wrapper with CSS-only transitions + IntersectionObserver. Animations replay when scrolling back.

| Block type | Animation | Duration | Rationale |
|------------|-----------|----------|-----------|
| Full-width sections | `fade-up` | 600–700ms | Gentle vertical reveal |
| Feature Right | `fade-right` | 800ms | Slides in from image side |
| Feature Left | `fade-left` | 800ms | Slides in from image side |
| CTA | `scale-up` | 800ms | Draws attention with scale effect |
| LogosBar | `fade` | 500ms | Subtle, non-distracting |

**Floating elements** use the `Float` component with infinite CSS keyframe bobbing (3.5–4.5s duration, staggered delays).

**Hover interactions** on FeatureCard, TestimonialCard, CardCta: `hover:shadow-[0_4px_24px_rgba(0,0,0,0.05)]` with 300ms transition.

All animations respect `prefers-reduced-motion`.

## Ordering Rules

1. Hero is always first
2. LogosBar immediately follows Hero
3. FeatureFrames alternate right/left
4. ValueProposition dark/light alternate as visual breaks between feature groups
5. Testimonials appear after the main feature showcase
6. FAQ and CTA are the final conversion blocks
7. Footer is always last
