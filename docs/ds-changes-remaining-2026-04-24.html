C2

Parser Webflow HTML → Dynamic Zone blocks

Requis Phase 1M · ~1.5 jour
Pourquoi Le contenu des 63 articles existe uniquement en HTML Webflow brut (colonne html_content dans Supabase). Il faut le transformer en blocks Strapi typés.
Comment
Script scripts/migrate/blog-articles-to-strapi.mjs :
1. Fetch 63 rows Supabase webflow_pages type='blog'
2. Parser HTML avec cheerio (ou jsdom)
3. Mapper les patterns détectés au sweep vers blocks Strapi (voir tableau mapping dans ds-audit-collection-gestion-projet.md §10)
4. Gérer les outliers (items 3, 4, 5) avec flags/skip
5. POST vers Strapi REST API (`/api/blog-articles`)
6. Log des articles skipped + raisons

Route template /[locale]/blog/[slug]

Requis Phase 1M · ~1 jour

Créer src/app/[locale]/blog/[slug]/page.tsx :
